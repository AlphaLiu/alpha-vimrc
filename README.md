###README Alpha-vimrc 
#####Introduction:
本项目目的是快速配置一个可用的vim环境以使vim更加方便易用。
本项目仍在不断更新中。

#####Installation:
0. Remove ~/.vim folder if already existed.
0. git clone git://github.com/AlphaLiu/alpha-vimrc.git
0. cd alpha-vimrc && ./install
